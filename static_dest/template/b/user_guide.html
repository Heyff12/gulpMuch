# coding: utf-8 储值教程

储值有什么物料？

  • 1桌牌和桌贴：桌牌和桌贴的二维码是提供消费者扫码进行储值的指定物料。
  • 2海报：定制化海报，既节省了老板设计的成本，也让店铺宣传更省心。

顾客怎么进行储值？

  • 1顾客可以通过桌牌，桌贴，海报上的二维码扫码进行储值。
  • 2顾客可以微信扫码支付时通过储值活动提示进行储值。

顾客储值充值后多长时间内可以使用储值？

  • 1顾客储值充值后立即就可以使用。

顾客怎么使用储值？

  • 1当顾客储值余额充足时，可以直接微信扫码选择储值支付即可（当前版本是余额充足时强制使用余额支付）。
  • 2当顾客储值余额不足时，可先让消费者用微信支付支付一部分，然后用剩余的储值余额支付。

顾客在哪里可以查看储值余额和储值消费？

  • 1顾客每次消费和储值都会收到短信通知。

怎么保障储值余额的安全呢？

  • 1初次储值充值时，请大家填写正确的手机号码，因为每次充值或消费时都有短信提醒，可及时核对储值余额，有任何问题可及时发现并联系店内工作人员。

储值余额是否有使用期限？

  • 1储值余额是否有使用期限，请注意查看店铺储值规则或者咨询店内小二。

储值充值的金额和储值消费可以退款么？

  • 1顾客储值充值的金额是不可以退款的。
  • 2顾客在使用储值余额消费时，只需和商家协商，并由商家发起退款，退款后交易的余额会自动退回储值余额中。

储值和储值消费怎么清款？

  • 1顾客储值充值后，储值充值的金额会在工作日的第二天到账（工作日不包括周六日）。
  • 2顾客储值充值后，使用储值余额消费时，此类交易是不计为实收的，因为储值充值的金额已经及时划款给商户了。
